# Niuanse Symfony

Serializer
==========

Grupy
-----

Trzeba zainicjować `MetaClassFactory` by grupy działały.

Workflow
========

Trzeba dodawać dokładnie jedno przejście `from` -> `to`, a jesli potrzeba `[from, from, from]` -> `to` to trzeba dodać wiele przejść o tej samej nazwie.
