# {{ $frontmatter.title }}

1.  Click `Windows+R`
2.  Type `Netplwiz`
3.  Uncheck `Users must enter a user name and password to use this computer.`
4.  Click `Apply`
5.  Type your password, and confirm password
6.  Click `OK`
