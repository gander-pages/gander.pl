`PhpStormLightEdit.desktop`
```ini
[Desktop Entry]
Name=PhpStorm LightEdit
Exec="phpstorm.sh" -e %u
Comment=A smart LightEdit for PHP and Web
Terminal=false
PrefersNonDefaultGPU=false
Icon=phpstorm.svg
Type=Application
```
